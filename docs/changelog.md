# Changelog

All notable changes to caswhois are documented here.

Follows [Semantic Versioning](https://semver.org/) and [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

### Added
- Initial release of caswhois WHOIS lookup service
- Domain, IP, and ASN WHOIS lookups
- REST API with JSON responses
- Web UI with dark mode default
- CLI client (`caswhois-cli`)
- Prometheus metrics endpoint
- GeoIP support (MaxMind GeoLite2)
- Built-in scheduler for automated tasks
- Tor hidden service support
- Self-update command
- Backup and restore functionality
- i18n support (en, es, zh, fr, ar, de, ja)
- PWA support (installable, offline-capable)
